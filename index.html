Stage

In februari 2021 moest ik de keuze maken waar ik mijn stage zou doen. Ik wou een leuke plek vinden met een toffe werksfeer, die dichtbij de deur gelegen is en waar ik mijn creatieve kanten kon laten zien. Zo ben ik bij Multimedium in Hoogstraten uitgekomen.

Ik ben de kennismaking spontaan aangegaan door het kantoor binnen te wandelen en dus vanaf de eerste moment, face-to-face, een goede indruk te kunnen geven van mezelf en mijn kwaliteiten.
Doordat ik al meerdere projecten in php had uitgewerkt tijdens mijn bachelor-opleiding kon ik, na dit kennismakingsgesprek en meerdere telefoontjes, mijn stage beginnen op deze toffe werkplek.

Tijdens de stage heb ik veel kennis opgedaan. Ik heb niet alleen beter leren programmeren met php, maar ik heb ook geleerd hoe er professioneel gecommuniceerd wordt in een bedrijf en met de klanten. Ik heb de tijdsdruk kunnen meemaken door te werken aan verschillende projecten met een deadline. Een website en database beheren en up-to-date houden zat ook in mijn takenpakket.

Dit traject heeft mijn zin in programmeren enkel versterkt. Ik hoop dan ook om dit nog veel te kunnen doen in de toekomst en dat ik in een toffe omgeving terecht kom met toffe collega's en waar ik nog veel meer kan bijleren.

RND-tool : Sterker op school

Bij Multimedium hebben ze soms het probleem dat de frontend developer niet verder kunnen zonder de backend.

Hiervoor moest ik een package schrijven waarmee de frontent developer json code kan schrijven om zo een "dummy" json-api te genereren met willekeurige data zoals gespecifieerd in zijn input-file.

De gegenereerde output kan op zijn beurt dan via javascript opgehaald worden.

Ook de backend developer kan zich via deze bestanden focussen op hoe hij zijn data moet meegeven.
Het input json-bestand kan via de package ook vergeleken worden met de output die de backend later zal geven voor compatibiliteit.

De broncode kan ik hiervan niet vrijgeven

WPF Minecraft management tool

Een project waarbij ik in groep een mooie WPF applicatie heb ontwikkeld dat werkt met een database.

Dit project heb ik samen met 2 medestudenten gerealiseerd.
Met deze applicatie kan men Minecraft servers aanmaken en de werelden ervan beheren.
Ook inventories van spelers kunnen aangepast worden.
'Recipes', 'advancements' en 'effects' van een speler in een server kan bekeken, toegevoegd en verwijderd worden.

Deze tool beheert geen echte minecraft servers en werkt met een zelf opgestelde database die de servers, werelden, spelers en de hardgecodeerde recepies, advancements en effecten bevat.

De applicatie werkt momenteel niet doordat de database niet meer bestaat, maar de broncode kan je nog op mijn github vinden.

Javascript 3d voorwerp weergeven op 2d canvas

In Javascript een canvas programmeren en de pixeldata bewerken op allerlei gekke manieren vindt ik leuk. Ik heb daardoor een projectje, dat ik eerder al een keer gemaakt had op scratch.mit.edu herprogrammeerd in een javascript canvas.

Het 3d object is opgebouwd uit meerdere veschillende 2d afbeeldingen die boven elkaar worden geplakt en gedraaid worden.
Zo lijkt het alsof er een 3d kast, donut, ballon of stoel op het scherm staat.

Er is ook een schaduw. Je kan het lichtpunt voor deze schaduw veranderen.
De schaduw werkt op dezelfde manier als het object zelf, maar is zwart en wordt een beetje uitgerokken.
